# Persian Question Answering System

An intelligent Persian (Farsi) question-answering system built with Python's HAZM NLP library. This project demonstrates advanced Persian text processing capabilities including stem-based semantic matching, intelligent scoring algorithms, and context-aware answer extraction.

## Features

- **Persian Text Processing**: Normalization, tokenization, stemming, and lemmatization
- **Semantic QA**: Advanced stem-based matching with tie-breaking rules
- **Intelligent Scoring**: Context-aware scoring with question-type detection
- **HAZM Integration**: Leverages Persian-specific NLP capabilities

## Technology Stack

- Python
- HAZM NLP Library
- Persian Language Processing
- Rule-based AI

## Use Cases

- Persian document analysis
- Educational Q&A systems
- Persian content search engines
- Language learning tools

### Installation
1. pip install hazm
2. pip install scikit-learn
3. **Clone the repository**
   git clone https://github.com/nahidsalehi/persian-qa-system.git
   cd persian-qa-system
4. source venv/bin/activate
5. python trained_qa.py


Perfect for demonstrating Persian NLP capabilities and intelligent text processing in non-English languages.
